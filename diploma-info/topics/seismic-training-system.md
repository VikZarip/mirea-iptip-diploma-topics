# Разработка информационно-обучающей системы для морской сейсморазведки

**Статус:** Зарезервирована

## Краткое описание (обязательно)
Создание обучающей системы для студентов МГУ по морской сейсморазведке. Система будет включать в себя возможность моделирования различных физических процессов (например, распространение волны в разных средах) и расчетные задачки для проверки, генерируемые ИИ.

## Ожидаемый результат (обязательно)
- Обучающие модели с возможностью изменения параметров
- Интуитивно понятный интерфейс
- Расчетные задачки по темам

## Возможные функции/компоненты (обязательно)
- Моделирование физических процессов.
- Интерфейс с разделением по темам (возможно с учебником).
- Расчетные задачки генерируемые ИИ.

## Технологические ожидания
- Python (matplotlib, scipy)
- PostgreSQL

## Метрики успеха
- Снижение количества ошибок при решении практических задач на 70%
- Среднее время сессии: > 25 минут
- Время загрузки страницы: < 3 секунд

## Основные use cases (обязательно)

- Студент выбирает тему, открывает модель. Читает справку (возможно). Вводит разные данные и смотрит результат.
- После тренировки с моделью, студент решает задачки по теме и вводит ответ.

## Научная новизна и актуальность

Система использует ИИ для генерации задачек, приближенных к тем, с которыми студент может столкнуться в реальной жизни, оживляя процесс.

## Методы, модели и данные

- Методы предоставленные МГУ.
- Сейсмические данные ЦАСД МГУ.

## Практическая польза (обязательно)

- Повышение интереса к учебному процессу студентов МГУ,
- Повышение эффективности обучения.

### Этапы проекта (план студента) — обязательно

- Анализ предметной области и сбор требований (сентябрь - начало ноября).
- Проектирование архитектуры системы (ноябрь).
- Разработка физических моделей (декабрь).
- Разработка пользовательского интерфейса (январь).
- Разработка задач генерируемых ИИ (февраль).
- Тестирование и оптимизация (март).
- Документация и статьи (апрель - май).

### Источники данных

- Сейсмические данные ЦАСД МГУ.
